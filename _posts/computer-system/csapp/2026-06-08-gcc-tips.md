# Linux Tools

A flag has a `-` symbol in front of it. All flags are options, but not all options are flags.

## GCC
### Prefix


### Flags
* `-c`: Compile and assembles the code.
* `-S`: Stop the compilation process before assembly.
* `-s`: Strip the symbol table from the compiled binary.

### Options
* `-m`: Specify hardware-specific or architecture-dependent compilation settings.
    * `-m32`: Generate code for a 32-bit environment.
    * `-m64`: Generate code for a 64-bit environment.
    * `-masm`: Generate an assembly file in specified syntax.
        * `-masm=att`: Generate ATT assembly. This is the default assembly dialect for x86_64 processors.
        * `-masm=intel`: Generate Intel assembly.

## GDB
### Abbreviations
* `b`
    1. Byte

* `x`
    1. Display
    2. Hex

---

### Commands
* `x/14xb myfun`: Display 14 hex-formatted bytes starting at the address where the function `myfun` is located.

## OBJDUMP
### Flags
* `-D`: Disassembles all sections of the file.
* `-d`: Disassembles executable sections of a binary file.
